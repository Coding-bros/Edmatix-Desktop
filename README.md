# Edmatix-Desktop

Initial Steps

## Download the App
## Open Terminal / CMD 
## CD into The Folder
## Then Follow the Commands Written Below

```
npm install
```

```
npm i -D electron@9.1.1
```
### then

```
npm i electron-is-dev@1.2.0
```
 
### Finally

```
npm start
``` 

### Which Should Open Edmatix Then You Can login Into Your Account
